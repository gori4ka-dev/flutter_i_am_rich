## О проекте

Это учебный проект **I Am Rich** — классическое "hello world" приложение для новичков во Flutter. Репозиторий пишется параллельно с видеосерией на YouTube-канале **Gori4ka**: каждый эпизод = отдельный шаг в разработке приложения, зафиксированный git-тегом или коммитом, на который можно посмотреть код именно на этом этапе.

**Кто ведёт:** backend PHP/WordPress разработчик с ~15-летним опытом, изучающий Flutter/Dart с нуля. Все объяснения — через аналогии с PHP/Laravel, без претензии на экспертность в мобильной разработке.

## 🚀 Как посмотреть код конкретного эпизода

```bash
git clone https://github.com/gori4ka-dev/flutter_i_am_rich.git
cd flutter_i_am_rich

# переключиться на состояние кода из нужного эпизода
git checkout 25c492d   # Ep.04
git checkout f26242c   # Ep.05
```

Если в репозитории заведены теги по эпизодам (например `ep04`, `ep05`), то удобнее переключаться так:

```bash
git checkout ep04
```
